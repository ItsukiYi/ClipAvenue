- [ ] 自动录制: 添加开播中的直播间 → 立刻开始录制
- [ ] 录制按钮: 未录时显示"开始录制", 录制中显示"终止录制", 不重复创建
- [ ] 文件管理器: 按主播分类→点击进入→列表/网格切换→点击播放→左滑删除

## 验证方法
1. QR 登录: 设置 → 扫码登录 → 扫描 → 确认 → 已登录
2. 录制: 添加直播间 → 检测 → 录制 → 仪表盘看指标 → 终止 → 文件管理看录播
3. 持久化: 添加房间 → 关闭App → 重新打开 → 房间存在
4. 文件管理: 录完后点"文件"tab → 看到分类 → 进入 → 点文件预览
