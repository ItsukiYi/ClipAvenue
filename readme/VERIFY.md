# 验收交接 — ClipAvenue

## 待验证 (最近改动)
- [ ] QR 扫码登录: 生成二维码后不会自动登出, 需扫码确认后才登录成功
- [ ] 录制: 点击录制后 3 秒内开始下载 (不再 15 秒延迟)
- [ ] 录制: 断线后自动刷新流地址重连, 不再用过期 URL
- [ ] 录制: 一个连接 = 一个文件, FLV 能正常播放拖动进度条
- [ ] 画质: 每个直播间能独立选择画质 (原画/蓝光/超清/高清/流畅)
- [ ] 配置持久化: 重启 App 后直播间列表/设置自动恢复
- [ ] 存储路径: 默认路径在 app 专属外存目录, 文件管理器可访问

## 验证方法
1. QR 登录: 设置 → 扫码登录 → 扫描二维码 → 确认 → 显示已登录
2. 录制: 添加直播间 → 检测 → 点击录制 → 观察仪表盘 → 录完后播放
3. 持久化: 添加房间 → 关闭 App → 重新打开 → 房间依然存在
